OpenCode JSON Maker

OpenCode 버전에 맞는 공식 스키마 규격을 생성합니다.

~/.config/opencode/opencode.json 

                    

                


                

                    
                    

                        
💡 TUI 적용 꿀팁:

                        
1. 위 JSON을 ~/.config/opencode/opencode.json에 저장하세요.

                        
2. 만약 설정 파일 수정이 번거롭다면, OpenCode TUI 창에서 직접 /connect 명령어를 치고 Custom Provider를 선택해 인터랙티브하게 설정할 수도 있습니다.

                        
※ 상단의 버전 토글을 통해 사용 중인 OpenCode 버전에 맞는 JSON 규격을 선택할 수 있습니다.
